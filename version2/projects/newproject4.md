## Latents Loop: Application Overview
- Recursive note-taking application that listens to active conversations.
- Intelligently updates a single markdown file in real time by understanding context.
- Unlike traditional transcription tools that just append text, it updates existing sections when circling back to previous topics.

## Tech Stack
- Utilizes Flask for the backend.
- Uses a template index.html file for the frontend, with plans to transition to Next.js in the future.
- Utilizes Grub for the core logic.
